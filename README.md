# Movie Rating Prediction with Python

This project predicts movie ratings using machine learning regression techniques.

## Features Used
- Genre
- Director
- Actor
- Release Year

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the program:
   python movie_rating_prediction.py

## Output
- Model evaluation metrics (MAE, RMSE, R2)
- Predicted rating for a sample movie
